# Battle Engine MVP

## Овощные Войны

---

# Общая концепция

Формат боя:

```text
3 vs 3
```

Бой проходит в реальном времени.

Игрок не управляет передвижением персонажей.

Персонажи самостоятельно:

* ищут цель;
* атакуют;
* используют обычные удары.

Игрок выступает в роли командира.

---

# Участники боя

Каждый игрок приводит в бой:

```text
3 персонажа
```

Например:

Игрок 1

```text
🥔 Картошка
🥒 Огурец
🍅 Помидор
```

Игрок 2

```text
🌽 Кукуруза
🌶 Перец
🍆 Баклажан
```

---

# Параметры персонажа

Каждый персонаж имеет:

```text
HP
Attack
Defense
AttackSpeed
AbilityCooldown
```

Пример:

```text
Помидор

HP = 100
Attack = 20
Defense = 5
AttackSpeed = 2 сек
```

---

# Состояния боя

```text
Created
↓
WaitingPlayers
↓
Countdown
↓
InProgress
↓
Finished
```

---

# Начало боя

После нахождения соперника:

Создается запись:

```text
Battle
```

Создается состояние боя в Redis.

Оба клиента подключаются к:

```text
BattleHub
```

Запускается таймер:

```text
3
2
1
БОЙ
```

---

# Игровой цикл

Battle Engine работает постоянно.

Шаг цикла:

```text
250 мс
```

Получаем:

```text
4 тика в секунду
```

---

# Обычная атака

Каждый персонаж имеет собственную скорость атаки.

Например:

```text
Картошка
каждые 2 секунды

Огурец
каждые 1.5 секунды

Помидор
каждые 2.5 секунды
```

Когда приходит время атаки:

персонаж наносит урон текущей цели.

---

# Формула урона MVP

Упрощенный вариант:

```text
Damage =
Attack - Defense
```

Минимальный урон:

```text
1
```

Пример:

```text
Attack = 20
Defense = 5

Урон = 15
```

---

# Выбор цели по умолчанию

Если игрок не отдавал команду:

персонаж выбирает ближайшую живую цель.

---

# Ручной выбор цели

Игрок может изменить цель.

---

## Один персонаж

Игрок нажимает:

```text
свой персонаж
```

затем

```text
вражеский персонаж
```

Получаем:

```text
🥒 → 🌶
```

Только Огурец атакует Перец.

---

## Несколько персонажей

Игрок выделяет область.

Все союзники внутри выделения становятся выбранными.

Пример:

```text
🥔
🥒
```

Затем игрок нажимает:

```text
🌶
```

Получаем:

```text
🥔 → 🌶

🥒 → 🌶
```

---

# Приоритет команды

Приоритет действует:

```text
до смерти цели
или
до новой команды игрока
```

---

# Суперспособности

У каждого персонажа есть одна суперспособность.

---

# Заряд способности

Во время боя персонаж накапливает заряд.

Например:

```text
1% каждую секунду
```

или

```text
+5%
за каждую атаку
```

Когда заряд достигает:

```text
100%
```

на фронте начинает светиться кнопка способности.

---

# Активация способности

Способность активирует игрок.

Например:

```text
Нажать кнопку Помидора
```

---

# Пример способностей

## Помидор

Томатная бомба

```text
Урон по площади
```

---

## Огурец

Скользкий манёвр

```text
50% уклонение
на 5 секунд
```

---

## Картошка

Земляной щит

```text
+30 защиты
на 5 секунд
```

---

# Смерть персонажа

Если:

```text
HP <= 0
```

то персонаж получает статус:

```text
Dead
```

Удаляется из списка доступных целей.

Больше не атакует.

Больше не получает заряд способности.

---

# Победа

Бой завершается если:

```text
все 3 персонажа команды мертвы
```

---

# Результат боя

Считается:

```text
Победитель

Разница HP

```

---

# Награды

Победитель получает:

```text
Семечки

```

если разница HP превышает порог:

```text
Выдать сундук
```

# Главный принцип MVP

Игрок не управляет движением персонажей.

Игрок принимает тактические решения:

* кого атаковать;
* когда применять способности;
* какую группу персонажей выделить.

Все остальное выполняет Battle Engine.
